# Homepage Admin Control - Quick Start Guide

## 🚀 Quick Start (5 Minutes)

### Step 1: Access Admin Panel
```
Go to: /admin/homepage
```

### Step 2: Edit Hero Section
1. Click "Hero Section" in left sidebar
2. Change title to your store name
3. Change subtitle to your tagline
4. Update button text and links
5. Upload featured image URL
6. Click "Save All Changes"

### Step 3: Edit Other Sections
Repeat Step 2 for:
- Elegance Section
- Features Section
- Categories Section
- Featured Products Section
- Newsletter Section

### Step 4: Enable/Disable Sections
- Toggle switch next to each section name
- Enabled sections appear on homepage
- Disabled sections are hidden

### Step 5: Reorder Sections
1. Find "Section Order" in left sidebar
2. Use ↑ ↓ buttons to reorder
3. Changes apply immediately

### Step 6: Preview
- Click "Preview Homepage" button
- Or visit `/` to see live homepage

## 🎨 Customization Tips

### Colors
- Use color picker or hex codes
- Example: `#ffffff` (white)
- Test on both light/dark modes

### Images
- Use HTTPS URLs only
- Recommended size: 1200x800px
- Format: JPG or PNG

### Text
- Keep titles short (under 50 chars)
- Subtitles can be longer
- Use clear, compelling language

### Buttons
- Use action words: "Shop", "Explore", "Learn More"
- Links: `/shop`, `/about`, or full URLs

## ⚡ Common Tasks

### Change Homepage Title
1. Go to Hero Section
2. Edit "Main Title" field
3. Save changes
4. Done! ✅

### Add New Feature
1. Go to Features Section
2. Click "Add Feature"
3. Enter icon, title, description
4. Save changes
5. Done! ✅

### Show More Products
1. Go to Featured Products Section
2. Change "Display Count" to higher number
3. Save changes
4. Done! ✅

### Hide a Section
1. Find section in left sidebar
2. Click toggle switch to OFF
3. Section disappears from homepage
4. Done! ✅

### Change Section Order
1. Find "Section Order" in left sidebar
2. Click ↑ or ↓ to move sections
3. Changes apply immediately
4. Done! ✅

## 🔧 Troubleshooting

### Changes Not Showing?
1. Click "Clear Cache" button
2. Refresh browser (Ctrl+Shift+R)
3. Wait 5 seconds
4. Check homepage again

### Image Not Loading?
1. Verify URL is correct
2. Check URL starts with `https://`
3. Try different image
4. Check image still exists

### Section Missing?
1. Check if section is enabled
2. Check section order
3. Clear cache and refresh
4. Check Firestore data
